5. In your own words, explain the box model. What is the "fix" for the box model, in other words, how do we make all elements respect the width we've given them? 

    The box model is how you can describe a box that wraps around an HTML element. There is content, border, padding and margin. Sometimes, the border can push the box further than you think. To fix that, use the box-sizing:border-box to include the padding and border within the element.
